# Store Permission Justification

## Google Play
Background location and physical activity are used to detect when a vehicle trip ends and save the parked location. Bluetooth is used only to identify user-registered vehicle devices. Notifications inform the user when a parking location is saved or a meter reminder is due.

## App Store
Location, motion, Bluetooth, notifications, and camera/photo access are requested only for parking detection, vehicle identification, reminders, and optional parking photos. Data is stored locally by default and is not used for advertising.
