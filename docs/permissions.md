# Permission Flow

Permissions are requested just-in-time.

1. Onboarding explains local-only parking storage and manual parking.
2. Manual save asks for while-in-use location.
3. Enabling automatic detection asks for motion/activity access.
4. Enabling background detection explains and requests background location separately.
5. Adding a Bluetooth vehicle asks for Bluetooth permissions.
6. Notifications are requested when reminders or automatic detection notifications are enabled.
7. Camera/photo permissions are requested only when adding a parking photo.
